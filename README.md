# oop2_10_sample
第10回 講義内資料サンプルコード

## 内容（メモ）
- データ(modelsとroutes。python)

    - 名前（入力）
    - 性別（選択）
  - 武器
    - 名前（入力）
    - 攻撃力（入力）
    - 武器タイプ（選択）

- 画面(templates。html) 
  - キャラクタ一覧 
    - キャラクタ制作画面
    - キャラクタ編集
  - 武器リスト
    - 武器制作画面
    - 武器編集画面
  - クエスト画面
    - クエスト追加画面
